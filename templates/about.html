About This Project

Welcome to FiveStat - The Football Data Analysis project!

This site is a project aimed at analyzing football fixtures and generating correct score heatmaps built from data-driven predictions.

This site will be evolving as I go, refining the model and adding new features.
I'm excited to see where we can take this!
